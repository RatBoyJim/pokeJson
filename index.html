A full-stack browser game based on the Pokemon trading card game

The project was designed to allow the following functionality:




  • Users can view a list of Pokemon supplied by PokeAPI

  • They are then able to select a specific Pokemon and view its full details

  • Users can battle each other with their chosen Pokemon

  • Uses a MongoDB database to store wins and losses

  • GoogleCharts utilised to display the wins and losses of each Pokemon used
